# 🌟 QuoteIt – Mood-Based Quote Generator

QuoteIt is a Python-based desktop application designed to uplift your mood with the perfect words at the right time. Whether you're feeling happy, motivated, calm, or even stressed, QuoteIt delivers inspiring quotes tailored to your emotions—paired with expressive GIFs for a more engaging experience.

---

##  Features

*  **Mood-Based Quotes**
  Get real-time quotes curated to match your current mood.

*  **GIF Integration**
  Enjoy expressive and relatable GIFs that visually enhance your emotional experience.

*  **Simple Desktop Interface**
  Built using Python and Tkinter for a clean and user-friendly UI.

*  **API Integration**
  Fetches dynamic content using external APIs for quotes and GIFs.

---

##  Tech Stack

* **Python**
* **Tkinter** (GUI framework)
* **APIs** (for quotes and GIFs)
